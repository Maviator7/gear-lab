# マニュアルトランスミッションの仕組み — 3D教育アプリ

車のマニュアルトランスミッション（MT）の動作原理を、3Dモデルとアニメーションで学べるWebアプリです。
「なぜギアを変えると力と速さが変わるのか」を、常時噛合式5速+リバースの簡略化モデルで可視化します。

## セットアップ手順

```bash
# 前提: Node.js 18 以上（18.x で動作確認済み）
npm install
```

> Node 18 環境で確実に動くよう、Vite 6 / React 18 / @react-three/fiber v8 / Tailwind CSS v3 に
> バージョンを固定しています（Vite 7 や Tailwind v4 は Node 20+ が必要）。

## 起動コマンド

```bash
npm run dev      # 開発サーバー（http://localhost:5173）
npm run build    # 型チェック + 本番ビルド（dist/）
npm run preview  # 本番ビルドのプレビュー
```

## 使い方

- **シフトレバー（H型ボタン）**: N / 1〜5速 / R を切り替え。クラッチ切断→シンクロ→係合のシーケンスが自動再生されます
- **クラッチトグル**: 手動でクラッチを切断すると動力が切れ、入力側が惰性で停止していきます
- **エンジン回転数スライダー**: 600〜3000 rpm（表示は約1/20のスローモーション）
- **初心者/詳細モード**: 部品名ラベル ⇔ 歯数・各軸RPMのライブ表示
- **マウス**: ドラッグで回転、ホイールでズーム

### 見どころ

- **ニュートラル**: 全ギアは回っているのに出力軸が止まっている（常時噛合・空転＝半透明表示）
- **シフト中**: シンクロリングがオレンジに発光し、カウンターシャフトの回転数が出力軸に合うまで変化する
- **リバース**: アイドルギア（18T）が間に入り、出力軸のRPMが負（逆回転）になる。歯数式 (18/12)×(30/18) でアイドラーが約分されて消える＝「比率は変えず方向だけ変える」ことが分かる
- **動力経路**: エンジン→クラッチ→入力軸→カウンターシャフト→係合ギア→出力軸がアンバーでハイライト

## アーキテクチャ

```
src/
  types.ts                 # 共有型（GearboxState / Sim / GearPairSpec ...）
  data/gears.ts            # 歯数・座標・ギア比（幾何整合性あり、変更時は docs/DESIGN.md 参照）
  data/explanations.ts     # 日本語解説テキスト
  hooks/useGearboxAnimation.ts  # シミュレーション本体（シフト状態機械 + 速度モデル）
  components/
    GearboxScene.tsx       # Canvas / ライト / 全パーツ配置 / ラベル
    Gear.tsx               # 歯付きギアメッシュ（ExtrudeGeometry、ジオメトリ共有キャッシュ）
    Shaft.tsx / Clutch.tsx / Synchronizer.tsx / ShiftFork.tsx / RotationArrow.tsx
    ControlPanel.tsx / ExplanationPanel.tsx
```

設計の詳細（符号規約・状態機械・歯数の幾何制約）は [docs/DESIGN.md](docs/DESIGN.md) を参照。
アニメーションは「可変Simオブジェクト + useFrameで直接メッシュ更新」パターンで、再レンダーなしに60fpsで動作します。

## 今後追加できる機能

- **車速・タイヤの可視化**: 出力軸の先にプロペラシャフト/ホイールを追加し、ギアごとの車速差を見せる
- **トルクメーター**: 各ギアのトルク増幅をリアルタイムのゲージで表示
- **エンジン音**: RPMに連動したピッチのサウンド（Web Audio API）
- **断面表示モード**: ギアボックスケースを追加し、クリップ平面で断面を見せる
- **クイズモード**: 「3速のギア比は？」「なぜRは逆回転？」などの理解度チェック
- **シフトミス体験**: クラッチを切らずにシフトするとギア鳴り（音＋振動表現）が起きる教材的演出
- **カメラプリセット**: 「クラッチに注目」「シンクロに注目」ボタンで自動カメラ移動
- **ダブルクラッチ/ヒール&トー**: 上級者向け操作の解説モード
- **多言語対応**: 英語UI切り替え
